# Website Deployment Instructions (GitHub Pages)

1. Create a GitHub Repository:
   - Go to https://github.com and log in.
   - Click "New Repository".
   - Name it (e.g., structured-cabling-site).
   - Set it to Public and click "Create Repository".

2. Upload Files:
   - Upload index.html, style.css, script.js, README.txt.
   - Commit changes.

3. Enable GitHub Pages:
   - Go to Settings → Pages.
   - Source: Deploy from branch → main → /root.
   - Save.

4. Access Your Site:
   - URL: https://<your-username>.github.io/<repository-name>/

Replace placeholder images and text with your own content.
